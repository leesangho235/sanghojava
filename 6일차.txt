변수
-기본형 : 정수, 실수, 문자, 논리
	변수안에 값이 저장
-참조형 : 나머지 다.(배열, 객체, 컬렉션...)
	변수안에 주소가 저장


변수는 이름과 타입 초기값을 다 넣어줘야 했는데
int a = 0;
int b = 0;
int c = 0; ...

배열은 그럴 필요가 없다.
int []    s;   -> 선언행위
배열자료형,배열의이름
s = new int[10]
배열의이름에는  -> 각 값의 주소가 들어있음 (참조)
배열의 장점
-하나의 이름으로 여러 저장공간을 선언
-타입을 한번만 선언
-초기값이 자동으로 지정-> 기본형(값 0),참조형(null)
배열의 단점

3명의 각 학생의 영어, 수학 점수입니다.
